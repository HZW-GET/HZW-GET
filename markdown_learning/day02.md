# Markdown 学习示例

## 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 段落
```markdown
这是一个段落。

这是另一个段落。
```

## 强调
```markdown
*斜体*
_斜体_
**粗体**
__粗体__
***粗斜体***
___粗斜体___
```

## 列表
### 无序列表
```markdown
- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2
- 项目 3
```

### 有序列表
```markdown
1. 项目 1
2. 项目 2
   1. 子项目 2.1
   2. 子项目 2.2
3. 项目 3
```

## 链接
```markdown
[GitHub](https://github.com)
```

## 图片
```markdown
![示例图片](https://example.com/image.png)
```

## 引用
```markdown
> 这是一个引用。
>> 嵌套引用。
```

## 代码块
```markdown
```
代码块内容
```
```

## 表格
```markdown
| 表头 1 | 表头 2 |
|--------|--------|
| 内容 1 | 内容 2 |
| 内容 3 | 内容 4 |
```

## 分隔线
```markdown
---
```
